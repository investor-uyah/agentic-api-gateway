📖 Quick Documentation

1. We integrate your public API into the Agent Gateway in JSON format.

"genderize": {
"base_url": "https://api.genderize.io",
"method": "GET",
"cache_strategy": "swr",
"ttl": 60,
"token": "****"
"bearer_token": false
},

Position your API safely behind our gateway infrastructure layer using a unique service name

2. Modify your API backend to reference our endpoint.

BASE_URL = "https://agentic-api-gateway.vercel.app"

response = await client.get(f"{BASE_URL}/v1/genderize?name={user_input}")

Requests to your API are routed through our gateway using the service name provided. The gateway automatically handles deduplication and cross-agent caching.

3. Your API traffic becomes

USER --> AGENT GATEWAY (Deduplication + Caching) --> YOUR API

The gateway automatically handles deduplication and cross-agent caching.

• Reduced API latency

• Efficient burst traffic handling
